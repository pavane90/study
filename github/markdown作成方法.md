# マークダウン言語の文法

readme.md の作成方法

# 0.改行

> マークダウン上で改行するためには文章の最後にスペースバーを２回、
> もしくは Enter を２回押下すう。

# 1.文字のサイズ

> 文字のサイズは＃を使って決める。
> 以下のように＃の数によってサイズが変わる

# アンニョンハセヨ

> '# アンニョンハセヨ'

## アンニョンハセヨ

> '## アンニョンハセヨ'

### アンニョンハセヨ

> '### アンニョンハセヨ'

#### アンニョンハセヨ

> '#### アンニョンハセヨ'

##### アンニョンハセヨ

> '##### アンニョンハセヨ'

###### アンニョンハセヨ

> '###### アンニョンハセヨ'

# 2. Bold

> 文字を Bold にしたい場合＊＊で囲む

俺が**ガンダム**だ。

> '俺が**ガンダム**だ。'

# 3. Italic

> Italic は\*で囲む

俺が*gundam*だ。

> '俺が*gundam*だ。'

# 4. 取り消し線

> 取り消し線は ~~で囲む

俺が~~ガンダム~~だ。

> '俺が~~ガンダム~~だ。'

# 5. コードの書き方

> 'コードは '''言語'''で囲む '

```c
#include<stdio.h>
int main(void)
{
printf("Hello Interface!\n");
return 0;

```
